# PowerBI-AdventureWorks-End-to-End

## 📌 Project Overview
This project is a comprehensive Business Intelligence solution developed for **AdventureWorks Cycles**, a global manufacturing company. The report transforms raw data into a professional interactive dashboard to track KPIs, analyze regional performance, and identify high-value customer segments.

## 🛠 Tech Stack & Implementation
* **Data Transformation (ETL):** Built automated workflows in **Power Query** for data cleaning, shaping, and creating a dynamic calendar table.
* **Data Modeling:** Established a robust **Star Schema** with 1:N relationships, active/inactive relationships, and optimized filter flows.
* **DAX Analytics:** Developed over 30 advanced measures using `CALCULATE`, `FILTER`, `RELATED`, and **Time Intelligence** patterns (MoM growth, YTD).
* **Interactive Visualization:** Designed a multi-page report featuring:
    * **Executive Summary:** High-level KPIs (Revenue: $24.9M, Profit: $10.5M).
    * **Customer Detail:** Profiling by income level (42% Low Income) and professional occupation.
    * **Product Performance:** Tracking Top 10 products by orders, revenue, and return rates.

## 🔍 Key Insights & Impact
* **Revenue Drivers:** Total revenue reached **$24.9M**, with a strong performance in the Accessories category.
* **Operational Risk Management:** Identified specific products with high return rates (e.g., Mountain Bottle Cage at **2.02%**), enabling quality control interventions.
* **Customer Segmentation:** "Skilled Manual" and "Professional" occupations were identified as the most active customer segments by order volume.
